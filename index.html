BabelGlove

The BabelGlove is a glove, embedded with sensors, that can translate the hand gestures of the wearer to a written and computer voice, using a computer program. It will be lightweight and comfortable for the user to wear. A fun, interactive program will be designed for beginning learners to create a more immersive experience. The glove will be fitted with flex sensors and an accelerometer. A microcontroller and a bluetooth module will be used to communicate between the sensors and the computer and software will take the data and produce an output on a desktop application. The entire system will be designed to be user friendly for all learning levels.
